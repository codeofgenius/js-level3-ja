# JavaScriptレベル3

JavaScript (JS) の基礎的な学習を修了した方に向けて、より発展的な文法を紹介します。変数、演算子などの詳細な動作について紹介します。このカリキュラムでは、ES2015以降の文法を基準として、JavaScriptの基礎的な文法を紹介します。

## ES2015（ES6）
ES2015（ES6）は、2009年のES5以来、約6年ぶりに標準化されたJavaScriptの劇的な大規模アップデートです。

背景には、ウェブアプリケーションの高度化により、従来のJavaScriptでは大規模開発や保守が困難になったことがあります。そこで、クラス構文、アロー関数、モジュール化、let/constによる変数宣言など、現代的で堅牢なプログラミングを可能にする機能が導入されました。

これ以降、JavaScriptは毎年更新される体制へと移行し、開発効率と言語仕様の安定性が飛躍的に向上しました。
